****************
YouTube Search
****************
The idea for this project comes from: 
	https://www.udemy.com/projects-in-javascript-jquery/#/

Project Details:
	Create a website where users can search & watch YouTube videos.

Hosted Link:
	http://pliu82.github.io/youtube-search

Instructions:
	Search for videos using the search bar.  YouTube will deliver them to the page.
	You can navigate back/forth between the pages of results.
	You can choose a different video screen size in the drop down menu.
	Currently each page of results will display 5 videos.
	When page is loaded for first time, default videos shown are of grumpy cat.

Languages: 
	HTML / CSS / Vanilla Javascript

3rd Party Tools: 
	Google's YouTube API V3

Helpful Resources (many thanks):
	- YouTube API documentation
	- Code Academy ( http://www.codecademy.com/en/tracks/youtube )
